# LondonBikeRides
Exploring London's bike sharing data using Excel, Python and Tableau.
The original dataset is from Kaggle and can be found here https://www.kaggle.com/datasets/hmavrodiev/london-bike-sharing-dataset.
After manipulating data with Python, I created a visualization dashboard in Tableau which can be found here: https://public.tableau.com/app/profile/kiana.alaei/viz/LondonBikeRides_17624133365340/Dashboard1.
